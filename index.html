TM Regetor 〜正規表現エディタ〜

ツイート

Input 正規表現

Target 置換対象となる文字列

Output マッチング結果

About

TM Regetor は, 正規表現の結果をリアルタイムで反映, 確認できるツールです.

Input に正規表現を入力すると, Target に Input 内容でマッチ判定をし 結果を Output に表示します.

また, 入力内容は URL の Hash に保存されるので, URL をコピペすれば周りの人と共有できます!!

正規表現を学ぶ際などに使ってくださいな♪
